# Travel Planner 🚗🗺️

This is a travel planning application that helps users find the shortest path between locations using **Dijkstra's Algorithm**. It features a visual frontend with interactive maps and a backend in C++ and Python.

## 🛠 Tech Stack
- C++ (Dijkstra’s Algorithm)
- Python (server integration)
- HTML, CSS, JavaScript (frontend)
- Leaflet.js and OpenRouteService API (maps)

## 💻 How to Run
1. Run `server.py` to start the backend.
2. Open `index.html` in a browser.
3. Choose source and destination to find the shortest path.


## 📌 Notes
- Make sure you have Python installed to run the backend.
- You need internet access for the map API to work properly.
